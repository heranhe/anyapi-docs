# 在线调试说明

## 在线调试

1. 点击 **“调试”** 按钮

![image.png](https://api.apifox.cn/api/v1/projects/2100343/resources/388494/image-preview)

2. 设置 OpenAI 的 api_key

![image.png](https://api.apifox.cn/api/v1/projects/2100343/resources/388496/image-preview)

该环境变量的 api_key 只保存在本地，不会同步到服务器，团队成员之间也不会相互同步。

![image.png](https://api.apifox.cn/api/v1/projects/2100343/resources/388499/image-preview)

3. 点击 **“发送”** 按钮，即可发起请求

![image.png](https://api.apifox.cn/api/v1/projects/2100343/resources/388502/image-preview)

## FAQ
1. 云端 Agent 错误
![image.png](https://api.apifox.cn/api/v1/projects/2100343/resources/388579/image-preview)
需要安装 Apifox 的插件：[Apifox Browser Extension ](https://apifox.com/help/applications-and-plugins/browser-extensions/microsoft-edge)

2. Invalid authorization header
![image.png](https://api.apifox.cn/api/v1/projects/2100343/resources/388580/image-preview)
需要在环境变量中设置 OpenAI 的 api_key

3. 无法请求地址 api.openai.com
![image.png](https://api.apifox.cn/api/v1/projects/2100343/resources/388584/image-preview)
需要梯子，即需要科学上网（且要开全局代理)
